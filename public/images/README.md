# Images Directory

Place your Cancer Compass images in this directory with the following naming convention:

## Image Files Needed:

### Homepage Carousel (ImageCarousel.tsx):
- `pickleball-fundraiser-group.jpg` - Large group photo at pickleball fundraiser
- `community-gathering-group.jpg` - Community gathering with volunteers
- `volunteers-group-photo.jpg` - Group photo of volunteers
- `pickleball-event-action.jpg` - Pickleball tournament action shot
- `fundraising-table.jpg` - Fundraising table with donations

### Gallery Page:
- `pickleball-fundraiser-group.jpg` - Fundraisers category
- `community-gathering-group.jpg` - Events category
- `volunteers-group-photo.jpg` - Volunteers category
- `pickleball-event-action.jpg` - Fundraisers category
- `fundraising-table.jpg` - Fundraisers category
- `pickleball-player-action-1.jpg` - Events category
- `pickleball-player-action-2.jpg` - Events category
- `pickleball-player-thumbs-up.jpg` - Volunteers category
- `volunteers-meeting-outdoor.jpg` - Volunteers category
- `group-with-money.jpg` - Fundraisers category
- `pickleball-court-group.jpg` - Events category
- `women-group-photo.jpg` - Volunteers category

### About Page:
- `volunteers-meeting-outdoor.jpg` - Founders/volunteers planning photo

## Image Requirements:
- Recommended format: JPG or PNG
- Recommended size: 1920x1080px or larger for carousel images
- Gallery images: Square format (1:1 aspect ratio) recommended, minimum 800x800px
- All images will automatically fallback to placeholder.svg if they fail to load

## Notes:
- Images are lazy-loaded for better performance
- Smooth fade-in transitions are applied automatically
- Images are optimized with proper object-cover for responsive design
