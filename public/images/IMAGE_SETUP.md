# Image Setup Instructions

## Quick Start

To add your images to the website, follow these steps:

1. **Navigate to the images folder**: `/public/images/`
2. **Upload your image files** with the exact names listed below
3. **Supported formats**: `.jpg`, `.jpeg`, `.png`, `.webp`

## Required Image Files

### For Homepage Carousel (5 images needed):
- `pickleball-fundraiser-group.jpg` - Main pickleball fundraiser with group
- `community-gathering-group.jpg` - Community gathering photo
- `volunteers-group-photo.jpg` - Group photo of volunteers
- `pickleball-event-action.jpg` - Pickleball tournament action shot
- `fundraising-table.jpg` - Fundraising table with donations

### For Gallery Page (12 images needed):
- `pickleball-fundraiser-group.jpg` - (same as carousel)
- `community-gathering-group.jpg` - (same as carousel)
- `volunteers-group-photo.jpg` - (same as carousel)
- `pickleball-event-action.jpg` - (same as carousel)
- `fundraising-table.jpg` - (same as carousel)
- `pickleball-player-action-1.jpg` - Pickleball player in action
- `pickleball-player-action-2.jpg` - Another pickleball action shot
- `pickleball-player-thumbs-up.jpg` - Volunteer celebrating
- `volunteers-meeting-outdoor.jpg` - Volunteers planning outdoors
- `group-with-money.jpg` - Fundraising success photo
- `pickleball-court-group.jpg` - Group at pickleball court
- `women-group-photo.jpg` - Group photo of volunteers

### For About Page (1 image needed):
- `volunteers-meeting-outdoor.jpg` - (same as gallery)

## Image Recommendations

- **Carousel images**: 1920x1080px (16:9 aspect ratio) or larger
- **Gallery images**: 1200x1200px (1:1 square) or larger
- **File size**: Keep under 2MB per image for best performance
- **Format**: JPG recommended for photos, PNG for graphics with transparency

## Note

If an image file is missing, the website will automatically show a placeholder. Once you upload the files with the correct names, they will appear automatically!
